## ¿Qué es uvpln?

`uvpln` es un equipo de **8 agentes especializados** para [Claude Code](https://claude.ai/code) enfocados exclusivamente en frontend. No es un agente genérico que hace de todo — es un especialista que conoce profundo el stack moderno y habla como la gente.

## El loop de calidad

El diferenciador de uvpln. Ningún componente es **listo** hasta que pasa el loop completo:

```
╔══════════════════════════════════════════════╗
║                                              ║
║   ui-architect  →  diseña el componente      ║
║         ↓                                    ║
║   ui-tester     →  lo rompe con browser real ║
║         ↓                                    ║
║   ui-architect  →  corrige con el reporte    ║
║         ↓                                    ║
║   ui-tester     →  APROBADO ✓                ║
║                                              ║
╚══════════════════════════════════════════════╝
```

## Instalación

```bash
# Clonar el repo
git clone https://github.com/jcarlosabc/un-viernes-por-la-noche.git
cd un-viernes-por-la-noche

# Instalar
bash install.sh
```

Eso es todo. Los 8 agentes quedan en `~/.claude/agents/` listos para usar con Claude Code.
