# uvpln — Plantillas de UI

Patrones visuales de referencia para ui-architect y ui-designer. No son código copiable — son **decisiones de diseño ya resueltas** que los agentes pueden usar como punto de partida.

Cuándo usarlas:
- ui-designer puede referenciarlas para proponer direcciones visuales
- ui-architect puede consultarlas cuando construye algo sin brief previo
- design-bridge puede compararlas con referencias externas del usuario

## Plantillas disponibles

| Archivo | Tipo de UI | Cuándo usarla |
|---------|-----------|---------------|
| `landing-page.md` | Marketing / conversión | Páginas de producto, portafolios, SaaS homepage |
| `dashboard.md` | Administración / datos | Paneles de control, analytics, backoffice |
| `auth.md` | Autenticación | Login, registro, recuperación de contraseña |
| `ecommerce.md` | Comercio electrónico | Catálogos, producto, carrito, checkout |
